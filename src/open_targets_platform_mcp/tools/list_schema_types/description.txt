List all types in the Open Targets Platform GraphQL schema.

This tool provides a high-level overview of the schema structure, showing
available Query entry points, Object types, Input types, Enums, and custom Scalars.

USE THIS AS THE STARTING POINT for RLM-based schema exploration. Instead of
fetching the entire schema (30,000+ tokens), start here to understand what's
available (500-1000 tokens), then use get_type_details() for specific types.

WORKFLOW:
1. Call list_schema_types() to see all available types
2. Identify relevant types based on user query
3. Use get_type_details(type_name) to explore specific types
4. Construct GraphQL query using the discovered structure

FILTERING:
Use type_filter to focus on specific categories:
- ["query"] - Only show Query entry points
- ["object", "input"] - Show Object and Input types
- Default (no filter) - Show all categories
